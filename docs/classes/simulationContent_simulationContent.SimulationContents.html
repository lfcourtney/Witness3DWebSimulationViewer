    • formatTag(tag: object): object

      Takes a JavaScript object produced by 'fast-xml-parser' parsing XML tag, where attributes and tag contents are separate, and reformats this object to make the attribute contents and tag contents part of the same object. For extra information, this is the same format that would be produced by the 'fast-xml-parser' if the 'preserveOrder' option was not enabled, which it is for this application
